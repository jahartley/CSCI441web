CSCI441 Project Requirements

Main Requirements

  • Get Data
  • Process Data
  • Save Data
  • Make decisions based on data
  • Send control commands
  • Execute commands

Basic Hardware Module Requirements

  • Connect via Ethernet or WIFI
  • Use a factory set ID for node deconfliction
  • Blink to locate and basic errors led
  • Use default network pin
  • Default network settings
  • Saveable user network settings
  • DHCP
  • DNS or IP addressing for MQTT server

Sensor Hardware Module Requirements

  • All Basic Requirements
  • Standardized sensor reading values by type of reading i.e.: temperature, humidity, pressure, frequency, energy (watts), current.
  • Ping or health check response
  • Only sends data on value change
  • Stateless system, listens for overall system restart message and sends all datapoints upon receipt with random delay.
  • Nodes for specific data points
    • Basic psychrometric air data point (temperature, humidity, pressure)
    • Refrigerant coil data point (inlet refrigerant temperature, outlet temperature, pressure, EEV setting)
    • Air Quality data point (basic air data plus PM, CO2, VOC, etc.)
    • Air Flow data point (measured air flow from fans, estimated flow from pressure differential, filter load or remaining life from pressure differential)

Control Hardware Module Requirements

  • All Basic Requirements
  • Generic Output i.e.: pin on/off, serial commands, RS485 commands, PWM commands.
  • Command Receipt Acknowledgement
  • Command Complete Acknowledgement
